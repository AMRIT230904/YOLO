🩺 Hemorrhage Detection using DIARETDB1 Dataset

## 📌 Project Overview
This project focuses on the detection of retinal hemorrhages using deep learning-based object detection models. The model is trained on the **DIARETDB1 dataset**, which contains retinal fundus images with annotated diabetic retinopathy lesions.

⚙️ Methodology
1. Data preprocessing and annotation conversion
2. Splitting dataset into train/validation/test
3. Training YOLO model on DIARETDB1 dataset
4. Evaluating performance using:
   - Precision
   - Recall
   - F1 Score
   - mAP (mean Average Precision)
   - ROC-AUC Curve
